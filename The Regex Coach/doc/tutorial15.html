The Regex Coach - Tutorial step 15

Select the "Split" tab.

You now see the result of splitting the target string with our regular expression. Each vertical bar is meant as a divider between two strings, i.e. in this case the result is the array consisting of the five strings "a", "ca", "caba", "" (the empty string), and "b".

Note that you can choose another divider with the radio buttons at the bottom if the vertical bar happens to be a part of your string. Also note that the "block" divider might slow down the whole program if you're working with very long strings.
